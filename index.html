1. How does JS work?

Javascript makes the website dynamic and it's a client side rendering scripting language. Javascript is not only high level language but also "just in time" compiled language which also supports dynamic typing, prototype-based object-orientation, and first-class functions. Javascript runs at the user browser which need to have js enabled. With the help of javascript event it calls the function and function complete the task. When we run js code inside web browser, it received by js engine and the engine runs the code.



2. What does javascript event loop do?

Javascript Event Loop execute the code , run the tasks and sub tasks and queued sub tasks. It mainly has a model based on an event loop. It is the main reason for Javascript asynchronous programming because javascript is single threaded language. Call stack keeps the track of all task in task. If task is completed then it is popped from the stack. There is also event queue which send new functions to the stack and it also process all task by sequence.
